# SUS — O Jogo do Impostor

Jogo multiplayer em tempo real onde jogadores entram numa sala, recebem informações secretas e tentam descobrir quem é o impostor através de blefe, discussão e votação.

## Tech Stack

- **Next.js** — Frontend e SSR
- **Convex** — Backend em tempo real (database, mutations, queries)
- **Tailwind CSS** — Estilização
- **Framer Motion** — Animações
- **Three.js / Shader Gradient** — Background visual

## Modos de Jogo

- **Modo Palavra** — Todos recebem a mesma palavra secreta, exceto o impostor
- **Modo Pergunta** — Jogadores respondem uma pergunta; o impostor recebe uma pergunta diferente
- **Modo Local** — Todos sabem o local, exceto o impostor
